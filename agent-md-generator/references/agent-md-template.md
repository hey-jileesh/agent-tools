## Quality bar

- Every command should be runnable as-written from the repo root.
- Prefer the repo's own scripts/targets over generic tool defaults.
- If you can't verify something, mark it `(inferred)` rather than stating it as fact.
- Keep it short. An agent re-reads this often; respect its context budget.
