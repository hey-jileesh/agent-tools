## Quality bar

- Annotations must be accurate — read the file before describing it.
- Include dependency versions; they inform which APIs an assistant should use.
- On updates, refresh the file index and dependencies, preserve human notes, and add
  a dated "Recent Changes" entry.
- Keep entries to one line each; this is an index, not documentation prose.
